碎片笔记

  • 浏览器一般包括渲染器和js脚本引擎

    Mozilla Firefox —Gecko
    Safari、Chrome —WebKit
    Opera —Presto
    Intenet Explorer —Trident

  • Js数据类型包括基本数据类型(原始类型)和引用数据类型(对象类型)；

  • 原始类型有：
    1.number
    整型；浮点型(注意精度问题)
    2.string
    放在引号里的大串字符序列：

    * 单双引号可以交替使用
* 总是记混的几个方法：slice(截取，类似于substring)、split[分割，返回数组]

    3.boolean
    任意的JS值都可以转换为布尔值

    * undefined,null,0,-0,NaN,""(空字符串)--->false
* 除以上外的所有值--->true

    4.null是object类型
    5.undefined未定义

  • 对象类型：Array,Function,Date,RegExp,Error

    对象是属性的集合，每个属性是有‘名/值对’构成；

    普通的Js对象是‘命名值’的无序集合;

    var arr = [1, 2, 3];
arr.length; // 3
arr.length = 6;
arr; // arr变为[1, 2, 3, undefined, undefined, undefined]
arr.length = 2;
arr; // arr变为[1, 2]
var arr = [1, 2, 3];
arr[5] = 'x';
arr; // arr变为[1, 2, 3, undefined, undefined, 'x']
大多数其他编程语言不允许直接改变数组的大小，越界访问索引会报错。然而，JavaScript的Array却不会有任何错误。在编写代码时，不建议直接修改Array的大小，访问索引时要确保索引不会越界.
实际上JavaScript对象的所有属性都是字符串，不过属性对应的值可以是任意数据类型。
JavaScript规定，访问不存在的属性不报错，而是返回undefined.
delete object.key;//删除对象属性
'key' in object;//检测是否拥有某一属性
要判断一个属性是否是object自身拥有的，而不是继承得到的，可以用hasOwnProperty()方法;

  • Tips:

    • 在内存中存储方式不一样：前者存储在栈内存(Stack)中，后者存储在堆内存(heap)中

    • 所以，前者值不可更改，后者可以

      类型转换：

      除了null和undefined不能进行类型转换外，其他所有类型的值都可以转换成其他类型。

    • string/number/boolean直接使用时，存在隐式转换；

    • toString()方法返回一个反映这个对象的字符串；
    • valueOf()方法返回对象本身；
